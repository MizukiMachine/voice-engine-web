# AI Voice Agent - Web Prototype
Web版プロトタイプ 最終要件定義書
1. AI会話・音声エンジン (VAPI連携)
起動方法: 画面上の「会話開始」ボタンをクリックしてセッション開始。終了ボタンで切断。
会話フロー: 開始後はVAPIのVAD（発話検知）により、ユーザーが話せば応答し、黙れば待機する自然な会話を実現。
Hotword (コマンド): 会話中に特定のキーワード（例：「撮影して」「録音して」）を検知し、即座に機能（Function Calling）を実行。
ウェイクワード: なし（今回は実装しない）。
2. スタジオ機能 & パーソナルメモリ (DB実装)
管理画面:
システムプロンプト（AIのキャラ・役割）の設定・保存。
音声ID、応答速度などのパラメータ調整。
長期記憶 (Memory):
ユーザーのプロフィール、好み、過去の文脈をデータベースに保存。
AIは会話時に常にメモリを参照し、パーソナライズされた応答を行う。
3. 体験シミュレータ (デバッグ機能)
GPS/ジオフェンス:
手動入力（場所名/座標）＋「到着」ボタンで、位置情報トリガー機能をテスト。
Push通知読み上げ:
手動入力（通知文言）＋「受信」ボタンで、通知読み上げ時の割り込み挙動をテスト。
4. デバイス連携
視覚 (Vision):
「撮影」コマンドで、Webカメラの映像をキャプチャしAIへ送信・解析。
録音:
「録音」コマンドで音声を別途保存。
5. 外部サービス連携
Google Workspace:
単一のテスト用Googleアカウントで認証。
Calendar/Docs: 読み取り（Read）および 書き込み（Write）の実装。
